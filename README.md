# hmview
Server based interactive heatmap visualization

## Requirements:
* git, c++11, make, wget

## Get source code:
```
git clone https://github.com/precimed/hmview
cd hmview
```

## Compile:
```
make
```

## Run:
```
./hmview
```
uses [example data](data/disreg_matrix_10x8.txt) by default
